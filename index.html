Cheat Sheet for HTML Tables

Table Tags – Making the Table

Tag Name Description
<table> Table Wrapper that defines a Table.
<thead> Table Head Set of rows that define the column headers in a table.
<tbody> Table Body Set of rows containing actual table data.
<tr> Table Row Defines a row in a table
<th> Table Header Cell Defines a header cell in a table.
<td> Table Data Defines a cell in a table.
<tfoot> Table Foot Groups the footer content in a table.

Table Attributes – Customising the Table

Attribute Name Description
colspan="#" Column Span Specifies that the data cell should span across # columns.
rowspan="#" Row Span Specifies that the data cell should span across # rows.
